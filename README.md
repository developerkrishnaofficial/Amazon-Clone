# 🛒 Amazon Clone - Front-end Project

A clean, responsive, and pixel-perfect front-end replica of the Amazon homepage built using HTML5 and CSS3. This project focuses on recreating Amazon's user interface while practicing responsive layouts, Flexbox, semantic HTML, and modern web design principles.

## 🚀 Live Features

### 🔹 Navigation Bar

* Amazon logo section
* Delivery location indicator
* Search bar with category dropdown
* Sign In & Account section
* Returns & Orders section
* Shopping Cart icon

### 🔹 Navigation Panel

* Today's Deals
* Customer Service
* Registry
* Gift Cards
* Sell
* Electronics Deals Banner

### 🔹 Hero Section

* Promotional banner area
* Amazon India redirect message
* Responsive content alignment

### 🔹 Shop Section

Includes 8 shopping category cards:

* Clothes
* Health & Personal Care
* Furniture
* Mobiles
* Beauty Picks
* Pet Care
* New Arrivals in Toys
* Discover Fashion Trends

### 🔹 Footer Section

* Back to Top button
* Multiple navigation columns
* Amazon branding section
* Copyright information
